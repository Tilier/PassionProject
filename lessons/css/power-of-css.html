Lesson #1: The Power of CSS

From: CSS

HTML is powerful, but CSS is too.

Here I’ve provided two examples: one example shows a website with no CSS, and one with CSS.


Without CSS:

https://www.w3schools.com/code/tryit.asp?filename=GFOOUM9Y5BNN


With CSS:

https://www.w3schools.com/code/tryit.asp?filename=GFOOLFA7HVUU


There is a big difference. Just based on these examples, you can probably see how important CSS is. Imagine how boring sites would be without CSS! They would always be displayed with a white background, a black serif font, ugly links, and overall just unattractive!

So now you can make sites look good, with CSS!

As you can see, CSS is a gamechanger, and I’m going to help you learn it. In the next lesson, I’ll explain the basics of CSS.
