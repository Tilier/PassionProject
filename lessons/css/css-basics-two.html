Lesson #3: CSS Basics - Part Two

From: CSS

CSS is made out of properties and selectors.

Here I’ll explain to you what properties are.

So now you know how to specify which elements you want to style. However, there’s no point in using selectors if you don’t use properties.

Properties are how you style elements. Each property has a property name, like color or width, and numerous property values. The property values can be numerical, like 7px, or they can be verbal, like blue.

Properties go inside selectors, and once you have specified all the properties for a selector, you show that you are done with that selector by closing it with }.

Properties always follow a certain format: property name: property value. For instance: color: white, font-size: 16px.

In most of the CSS lessons, we’ll discuss different kinds of properties.
