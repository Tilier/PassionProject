Lesson #1: Something something

From: HTML

HTML is powerful.

It rules all. Without it, you wouldn’t be on this site reading this. In fact, online school would probably be done for. Every site you see is made with HTML. Every. Single. One. You may ask, “what about website builders, like Wix, Shopify, or SquareSpace?” Well, those website builders only make it easier. Normally, you would have to program a site in HTML, probably CSS, and possibly PHP, Javascript, and JSON. These website builders only generate all of the code (plus possible SQL) for you, so that you don’t have to write the code.

Here’s an example — right-click anywhere on this site. Then hit “inspect” or, for Safari, “inspect element.” A bar will appear, probably on the bottom or right side of the screen, showing all of the code that went into that site. So, you can see all of the code I wrote to build this page.

Then, go to photographyphiles.com, a site built with Wix, and inspect. All of the code you see was generated by Wix — but it’s still in HTML, CSS, etc.

Basically, without HTML, there would be no sites.

But what about CSS? Javascript? PHP?

They both need HTML to survive. They both use what was coded in HTML.

In the next lesson, you’ll learn some basics of HTML.
