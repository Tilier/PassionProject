Lesson #3: Types of Tags

From: HTML

Now that you've learned about tags, lets learn some tags!

Here’s a list of HTML tags:

  • !DOCTYPE html - specifies that the document language is HTML, required
  • html - specifies what is in the website, required
  • body - all the content in the site goes here, required
  • head - all the info about the site goes here, e.g. the favicon and the title
  • !-- ... -- - adds a comment between the <!-- and the -->.
  • div - specifies a section (more for usage in CSS)
  • a - on click redirects to link specified using href attribute, use target=”_blank” to open in new tab
  • button - creates a button, and unlike a, padding is treated as a solid element
  • h1, h2, h3, h4, h5, h6 - depending on number will set all the text inside to a certain font size (the smaller the number the bigger the font size)
  • p - specifies a paragraph
  • em - italics
  • br - line break, also 18px high
  • strong - bold, sometimes doesn't work well depending on the font
  • img - self-closing tag, displays an image
  • video - displays a video
  • iframe - displays a webpage on your website, some websites will not allow it though
  • span for CSS styling, gives all the elements inside default inherited styling, like color. (More about span in the learning CSS section.)
  • form specifies that everything inside it is part of a form
  • input allows user input through different types e.g. text and radio (form lesson coming up soon, also self-closing tag)
