Lesson #2: HTML Basics

From: HTML

First, we’ll begin with the basics.

HTML is built using tags. An example of a tag is h1. Another is div. Inside most tags, you can put content. For example, inside a div you can put an h1. However, you also have to close these tags. Once you put all of the code you want in a tag, you have to end it. Here’s an example:

In this, I place an h1 element inside a div element. As you can see, inside the h1 tag is the word “Hello.” This word will appear in a big, bold font due to the default styling for h1. However, h2 would be smaller, then smaller for h3, etc. View this example here:

Now notice how once I am done with the text I want in the h1 tag, I close it with /h1. For h2, I close it with /h2, and so on and so on. But what would happen if I didn’t close the tags?

The h2 seems to appear before h1. Everything gets mixed up, since the h2 is now inside the h1 tag, and the system just gets confused. This is what the code is interpreted as:

<div>
    <h1>
        Hello!
        <div>
            <h2>
            Hello!
            </h2>
        </div>
    </h1>
</div>

So what would it look with actually closed tags?

In the end, close your tags if they need to be closed (and most tags do).



Lastly, you should know what parent elements are. Parent elements are what enclose its children elements. or instance, if you have this code:

<div>
    <p>Hello!</p>
</div>
